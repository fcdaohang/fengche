长文预警：

DexGuru DAO 即将推出！#

从一开始，我们就致力于产品优于一切的法则。我们的目标一直是建立一个人们真正想要使用的强大平台，然后由$GURU代币提供支持，开始将治理权力和资金管理交给社区。

我们的目标是尽可能分散地启动DAO治理，同时保持社区的参与和娱乐。我们希望社区成员将自己视为项目的一部分，与相同的目标保持一致。因此，社区应该向前迈进，参与DAO Bootstrap阶段和治理挖掘，而不仅仅是盲目地遵循核心团队的决定。在此期间，我们将寻求所有与DAO相关的重要议题并达成共识。

在2022年2月1日之前向DexGuru付过小费的早期用户将能够铸造GURU Season Pass NFT，允许他们在DAO引导阶段参与治理挖矿。之后，将由DAO决定何时以及如何启动$GURU代币，以及如何奖励早期治理参与者。

我们预计DAO前期投票需要一个月左右，铸币之后，TGE之前。但也可能有变。

谁可以参加？#

我们认为，DAO的基本规则和结构应该由一个真正希望在DAO治理中拥有发言权且高度敬业的社区来定义。这就是为什么我们定义了一个邀请参加DAO引导的列表：

在 2022 年 2 月 1 日之前支付 DexGuru 小费并早期支持我们社区的用户，哪个公链和给予小费的数量并不重要。如果您在2022年2月1日之前向DexGuru支付过小费，则有资格铸造GURU Season Pass NFT。

LobsterDAO NFT 持有者 DEXT代币持有者 UniWhales代币持有者 Chartex代币持有者 AstroTools代币持有者

上述地址被列入白名单，因为他们可以成为特别有价值的社区成员，具有相关知识，并且已经有与他们的地址相关的重要活动。上述用户的地址已列入白名单，他们将能够免费铸造GURU Season Pass NFT。该NFT将允许社区成员参与DAO引导阶段，并对DAO前的决定进行投票，例如何时以及如何部署$GURU代币，早期参与者应该获得什么奖励等。

GURU Season Pass NFT mint#

mint暂定于3月底在以太坊上发行。白名单将在mint之前在Github上发布。

mint是免费的，只需支付gas费用。

mint将在短时间内开放 - 大约3天左右。Season Pass的重点是在启动$GURU代币之前参与DAO Bootstrap阶段。长时间开放没有多大意义。

为什么要参与 DAO 引导阶段？#

DAO引导阶段的主要思想是为社区提供从一开始就塑造DAO的权利和手段。Season Pass NFT持有者将不得不参加多个治理民意调查，并就多个主题发表意见，包括如何向早期参与者分发代币。如果您要铸造Season Pass，您必须准备好积极参与决策。

基于对多个DAO的研究，有几个重要的主题定义了DAO，它们是必不可少的：

DAO的使命、价值观和宣言 治理和决策过程 组织结构和规则 $GURU 代币经济学 社区任务、目标和指标 所有这些主题都需要在DAO前阶段的研讨会和Discord/Discourse中讨论，然后形成提案进行投票。Season Pass NFT持有者在此阶段做出的决定将形成DAO基本规则和机制。

请记住，铸造Season Pass并不能自动保证$GURU空投。您必须参与多个治理民意调查，并就多个主题陈述您的意见，包括如何向早期参与者分发代币。

治理#

在 DAO Bootstrap 阶段，将为社区提供用于创建提案和投票的工具。

投票过程将基于链下快照。只有Season Pass NFT持有者才能参加投票。快照将在 NFT mint过程完成后启动。

主要通信渠道是GURU Season Pass NFT持有者和话语服务器的私人Discord频道。链接将在稍后公布，随即开放mint。

为使治理过程有效，建议采取以下措施：

以表情符号投票的形式在Discord进行初步讨论和预热 根据讨论情况，社区在Discourse上发布正式的提案 提案草案发布后，有3天的时间来讨论并就该主题发表评论。根据这一讨论，对该提案进行修正 更新后的最终提案将发布在快照上以进行投票。投票期为3天。在期限结束时，将以少数服从多数作出决定 一旦我们在所有主要议题上达成共识，我们将宣布DAO的发布日期。目前的预期是2022年5月底

常见问题#

如果DAO决定为自己做空投 / 入侵纳尼亚（开玩笑） / 殖民火星（夸张了）怎么办？#

Guru Season Pass NFT 将允许那些mint它的人在DAO引导阶段参与治理挖矿。

请自己研究什么是治理挖矿。在此处阅读有关治理挖掘理念的更多信息 请自己研究DAO是什么。请了解这不是空投。 之后，将由DAO决定何时以及如何启动$GURU代币，以及如何奖励早期治理参与者。

如果人们出售NFT怎么办？他们的新所有者是否有资格进入引导阶段？#

Season Pass NFT所有者可以投票并参与DAO引导阶段。请注意，仅持有Season Pass NFT并不能保证持有人$GURU代币空投。

该团队是否有铸造NFT的计划，因此他们在DAO之前拥有投票权？#

不，团队不会铸造代币。我们可能有一两个，因为我们中的许多人使用个人钱包进行测试，但没有什么重要到足以影响投票结果。

我通过使用您的DEX来支持您…/这是不公平的…/您欠我的，因为我免费使用了您先进的链上分析工具…#

通常，我们不想在后端存储有关用户的任何信息;我们不想成为用户数据/个人资料的守门人。我们使用 0x API 进行订单路由。没有流动性池或DexGuru AMM。从这个意义上说，我们只是一个透明地传递链上订单的dex聚合器。因此，小费机制是我们检查链上的唯一方法，如果您曾经在我们的平台上进行交易。

我们感谢您为追溯性地参与社区生活所做的努力。但这不会影响我们对Season Pass NFT的决定。

不会有不同的等级或任何类似稀有的东西。任何在2月1日之前给小费的用户都可以铸造1个（相同的）NFT（用于1个钱包）。无论您使用DexGuru多长时间或您给了多少小费，我们都希望为每个人提供参与DAO Bootstrap阶段的平等机会。

请不要将这种创建DAO的方式与空投混淆。
