今天，Telcoin 很高兴地介绍去中心化金融平台期待已久的“V3”。我们将展示两个新网站，详细介绍 Telcoin 平台的两个核心层，以及如何参与的说明。

V3 不仅仅是一次增量更新——它标志着 Telcoin 的一个重要进化里程碑。我们开始了专注于普惠金融的旅程，我们相信 V3 将这一使命推向了新的高度。 Telcoin 不仅将提供地球上最实惠的汇款网络，24/7/365 全天候即时全球结算，而且我们已将平台的流动性引擎分散给用户，使他们能够参与并收取他们现在专属的费用从他们在日常生活中依赖的产品中奖励传统金融机构。 Telcoin V3 标志着一个新的、用户拥有的、面向普通人的金融系统的诞生，我们非常高兴能成为我们认为将对数十亿人的生活产生积极影响的一小部分。

我们仍然致力于将以太坊作为最开放、最通用和被广泛采用的区块链。同时，在 V3 中，我们认识到汇款和日常支付需要能够负担得起的交易。随着可扩展性战争的激烈进行，我们已经密切关注各种解决方案，这些解决方案将使我们能够利用以太坊的网络效应和安全功能，同时为我们的客户提供快速且负担得起的用户拥有的产品，而不会影响去中心化。

经过深入分析，我们决定在 Polygon POS 上发布 Telcoin 金融平台 V3，并利用 Polygon SDK 启动 Rivendell 网络，这是一条由 GSMA 正式会员移动网络运营商作为独家授权验证者的以太坊侧链放。

让我们深入了解我们今天究竟要推出什么——以及我们打算在未来几周和几个月内推出什么。

Telcoin 平台#

Telcoin V3 是一个用户拥有的去中心化金融平台，由活跃的 Telcoin 用户和合作伙伴提供支持。 Telcoin 平台是一个多层的全栈架构，在每一层的需求和供应方都有独特的市场参与者，具有特定的角色和职责。

Telcoin 平台堆栈由三个主要的互补层组成——Telcoin 应用程序、TELx 和 Rivendell——将在下面更全面地解释。

Telcoin应用程序V3#

Telcoin 应用程序是 Telcoin 平台的合规、安全且易于使用的界面。下一代 Telcoin 应用程序将在 Polygon 网络上推出，具有多种新功能，我们非常自豪地宣布推出两款产品：

Send Money Smarter (SMS) Network 是一个用户拥有的去中心化国际汇款平台，颠覆了 7000 亿美元的汇款行业。 SMS 将外汇分散到 DeFi 协议上的活跃 Telcoin 用户，使汇款人能够以接近中间市场价格的即时全球结算，以一小部分成本获得现在专门授予传统金融机构的费用。#

TELxchange 是一个用户拥有的、去中心化的数字资产交换网络，以 TEL 作为储备资产和交换媒介。 TELxchange 通过 DeFi 协议将代币交换分散给活跃的 Telcoin 用户，使用户（在受支持的地区）能够直接从他们的银行账户在 TEL 和 TELx 生态系统资产之间进行交换，同时获得现在专门授予中心化交易所的部分交易费用——无需给予保管他们的资产#

这两种产品都是由活跃的 Telcoin 用户在 DeFi 协议或 TELx 上唯一启用的，并且随着每个去中心化市场的可行性，应用程序中的新最终用户功能将逐步采用。一旦第一批市场准备好进行整合，Telcoin 将推出具有上述功能集及更多功能的应用程序 V3。

介绍 TELx#

TELx 是 Telcoin 平台的去中心化流动性网络，是 Telcoin 的自托管金融产品套件的引擎——我们今天将展示其中的两个。

在规模上，所有 Telcoin 应用程序交易量将专门通过支持 DeFi 协议的 TELx 市场流动。该系统旨在实现真正的用户所有权，活跃的 Telcoin 用户为网络提供供应方服务，为他们在日常生活中所依赖的产品提供动力——同时收取通常专门授予金融机构和平台原生资产 TEL 的费用.

我们将在两个 DeFi 协议 Quickswap 和 DFX (Polygon) 之上在九个去中心化市场推出 TELx 激励措施，这两个协议将专门为下面详述的两种新的用户拥有的产品提供动力。新的 TELx 市场的添加遵循基于规则的方法，随着用户使它们具有足够的流动性和去中心化性以达到最终用户的生存能力，它们最终将集成到 Telcoin 应用程序中。

介绍 Rivendell#

Rivendell 是一个开放的以太坊侧链，由 MNO 运行、验证和保护。 Rivendell 使移动网络能够为其订户提供基于区块链的自托管金融服务——无需成为银行或托管基金，其成本仅为传统系统的一小部分，同时利用其现有网络和欺诈基础设施来支持系统。

TELx Genesis 产品#

除了展示我们的 TELx 市场层之外，我们还很高兴地宣布在它之上推出的前两种产品：Send Money Smarter (SMS) 网络和 TELxchange。

SMS网络是用户拥有的全球跨境汇款网络，为 Telcoin 数字汇款的下一次迭代奠定了基础。

Telcoin 发行开始#

随着 TELx 的推出，正式的 TEL 发行将根据 TELx 网站“关于”部分新找到的文档开始分发。随着我们推出平台的每个新组件，相应类别的发行将开始分发给提供发行激励服务的市场参与者。更多关于发行的文件将在未来几个月内发布，并将发布在 Telcoin 网站上。

为了启动 TELx，我们将在 Polygon 上的两个受支持的 DeFi 协议中推出九个激励市场：Quickswap 和 DFX。在 TELx 门户中质押其 LP 代币的用户将获得 TEL，并且在某些情况下，将有资格获得双重奖励——同时获得 TEL 和 DeFi 协议的原生奖励代币。
