前言#

在每晚丧心病狂的搜集信息了解项目以及介入一些资本之后，我在过去几年中收到了一些最好的空投，包括： $UNI、$ENS、$dYdX、$INCH、$SNX、$GFI、$ILV、$ICP、$NOTE、$INST

本指南中，您通过简短列表可以找到一些于2022年可能空投的最重要项目。指南还附带了一些如何获得它们的操作，但要有一些基础知识。

Airdrop空投是一种新的增长黑客技术，它允许项目通过承诺未来的奖励来吸引（并保留）早期用户/布道者进入生态系统。这些项目不是分配营销预算，而是建立未来代币的份额，以奖励过去的用户，分散产品并吸引新人。

说白话就是免费送馅饼，吸引大家关注。话不多说，开始：

1. Hop Protocol#

最有前途的 ETH <-> L2 桥之一。我们生活在一个多链世界中，因此在未来几年桥的机会将是巨大的。 操作：

  • 不同链之间的资产跨链操作
  • 在没有 IL 的池中提供流动性和质押资金 APP地址 风车注：APP目前有四个功能：Send Pool Covent Stake，对应跨链发送资金、提供流动性、资产转换、质押，建议都可以操作一下，选个Gas费低的时候

2. Connext / XPollinate#

@ConnextNetwork / @xpollinate_io EVM 链之间的另一个重要桥梁。 操作：

  • 不同链之间的资产跨链操作 APP地址

3. Element Finance#

用于固定和可变收益市场的开源协议。 操作：

  • 以固定利率投资一些钱
  • 为资金池提供资金
  • 使用 Aztec 版本 APP地址

4. ECO#

密切关注@andy_bromberg 和团队正在构建什么来彻底改变金钱的概念。他们总是发#KYEP 操作：

  • 通过 Twitter @eco_accountant BOT 与社区互动，并成为 Discord 中的活跃成员。 APP地址

5. Argent#

最好的以太坊钱包之一，具有最佳的用户体验，并且已经与许多产品和 L2 集成。 操作：

  • 在主网和 ZkSync 上激活帐户并开始使用一些协议。
  • 还可以使用 Starkware 的第一个 Web 浏览器钱包。 APP地址

6. Aztec Network#

Aztec 是货币互联网的终极安全屏障。使用零知识证明构建的隐私网络 http://zk.money 允许用户隐蔽 ETH 操作：

  • 从一个钱包中隐蔽一些 ETH，然后发送到另一个钱包
  • 将它们保留在网络中并等待Element部署 APP地址

7. Foundation#

旨在建立一个新的创意经济平台(NFT)，类似于 Opensea 操作：

  • 购买一些您喜欢的艺术品并以新的方式支持艺术家！（空投类似于在本平台交易的早期用户，参考ENS） APP地址

8. Polymarket#

Polymarket 是一个去中心化的信息市场平台，旨在揭开对您最重要的现实世界事件的神秘面纱。它适用于 Polygon，因此使用起来很便宜。 操作：

  • 为你喜欢的事情下注（参与）
  • 为高APY市场提供流动性（注意风险） APP地址

9. Superfluid#

流支付赛道著名项目，处理订阅、工资、奖励和任何价值流，每秒持续结算。 操作：

  • 使用APP发送money流，加入他们的Discord，并继续使用它 APP地址

10. Zig Zag Exchange#

它是第一个在 zkSync 上运行的具有中央订单簿的 DEX。 操作：

  • 将一些资金桥接到@zksync 并进行一些交易！ APP地址

11. OPYN#

Opyn 正在构建 DeFI 原生衍生品和期权基础设施。现今最好的 DeFI 协议之一。 操作：

  • 直接投 oToken（看跌或看涨期权）
  • 购买并持有 oToken
  • 开一个多头仓位 APP地址

12. Ondo Finance#

Ondo 是一家提供结构化产品的去中心化投资银行。通过@feiprotocol，他们为那些想要租用流动性的 DAO 开发了一个解决方案。 操作：

  • 在 Ondo Vault 上存款 APP地址

13. ZORA#

在以太坊上购买、出售和管理 NFT 的开放协议。 操作：

  • 买一些你喜欢的 NFT
  • 创建、管理和销售 NFT
  • 铸造 Zorb 并成为积极的参与者 APP地址

14. RENFT#

为 NFT 构建金融层。他们推出了第一个通用租赁市场，并建立了第一个无抵押贷款/借款池。 操作：

  • 成为 Astrocat 持有者
  • 使用其协议 APP地址

15. MIRROR#

用于共享和资助任何东西的基本 web3 工具包。从写下你的最新想法，到为下一个大 DAO 建造家园。 操作：

  • 开始写文章
  • 开始众筹活动
  • 创建您的第一个版本
  • 启动您的 web 3 节点 APP地址

16. Rabbit Hole#

通过发现和使用最新的加密应用程序来赚取加密货币。这是新人发现新事物和边做边学的最佳方式。 操作：

  • 完成所有技能并获得 XP
  • 完成所有任务
  • 铸造冒险徽章 APP地址

17. Zed Run#

ZED 是一款基于区块链技术的可证明公平的数字赛马游戏。通过建立一个星光熠熠的获胜赛马马厩来创造辉煌。 操作：

  • 有一个马厩和一些马
  • 参加了一些锦标赛事 APP地址

18. Euler Finance#

Euler 是一种资本效率高且无需许可的借贷协议，可帮助用户从其加密资产中赚取利息或对冲波动的市场 操作：

  • 通过协议借款 APP地址

19. Slingshot#

  • 在每条链上部署的App内进行交易
  • 尝试 NFT 市场 APP地址
