项目白皮书：#

Ziktalk Web 3.0短视频平台 白皮书 v 2.0 摘要：Ziktalk 是 Web 3.0 短视频平台，拥有超过 65 万用户，允许内容创作者和消费者通过社交挖掘的概念赚取 ZIK 代币在应用程序内进行交互并奖励有利于生态系统发展的贡献的用户。Ziktalk 为 98% 收入低于贫困线的内容创作者和发展中国家的用户提供了变现机会。

一、简介#

问题#

互联网的快速发展和智能移动设备的普及使得在线社交媒体网络为广大人群所用，We Are Social 报告了全球 38 亿社交媒体用户 [1]。多年来，Facebook、YouTube、Instagram、TikTok 和 Twitch 已成为用户分享故事和提供娱乐内容的流行平台。 虽然这些服务的用户可以通过他们的帐户和内容获利，因为他们越来越受欢迎以获得奖励，但这些平台无法实现即时和直接的用户对用户的补偿。为了奖励创作者，用户必须将自己暴露在不需要的广告或推广内容中，或者他们必须从平台购买现金或物品以发送给他们最喜欢的创作者。通过这种方式，用户必须依赖平台方能促进用户间交易的便利。

我们的解决方案#

一个使用户能够直接对彼此内容进行奖励的平台，不但可以激励积极的网络参与度，同时还确保用户的价值贡献得到适当的补偿。Ziktalk 是一个短视频平台，允许用户发布内容并因其受欢迎的程度而获得奖励。

Ziktalk 的核心是 ZIK 代币，这是一种基于 Luniverse 公链的代币，它为整个Ziktalk网络提供动力，可以通过称为社交挖矿的过程获得。 用户将获得免费代币来完成诸如注册帐户、推荐朋友、登录等任务。然后，这些代币将用于奖励生成有趣内容的其他用户。ZIK 代币形式的价值转移几乎是即时的，交易费用为零，以最大限度地减少价值磨损。因此，用户将能够在 Ziktalk 生态系统内直接使用他们免费获得的代币相互奖励。

二、Ziktalk#

Ziktalk 不仅仅是一个想法，它是一个拥有真实用户的推出产品： Luniverse 的 2021 年年度报告将Ziktalk 评为其区块链上的顶级 dApp，拥有 650,000 名用户和 5,462,579 笔主链和侧链上的交易，在 Luniverse 区块链上的所有 1,184 个 dapp 中记录了最多的交易。 凭借 650,000 名用户、5,462,579 笔交易，Ziktalk 在 Luuniverse 区块链上的 1,184 个 dApp 中排名第一。真实用户，真实数字

专注于短视频#

据估计，到 2023 年，移动互联网连接总数预计将达到 34 亿，短视频预计将出现显着增长。在 Ziktalk 中，所有用户交互都是通过从创世视频到回复的短视频进行的。Ziktalk 完全消除了文本的使用，成为一个鼓励和奖励创意内容创作的平台；只关注最有前途的视频格式。 短视频和垂直视频非常适合蜂窝设备内容消费。Ziktalk 是一个平台，内容创作者仅在短视频垂直视频中进行互动，并发现制作垂直视频和短视频的新方法。

Web 3.0平台打造创客经济“中产阶级”#

Web 2.0 社交平台主要分为三个部分平台、内容创作者、内容消费者——平台占据广告收入的最大份额，一小部分内容创作者通过赞助创造财富，以及内容消费者向平台提供个人数据以换取免费内容。尽管社交媒体和创作者经济快速增长，但社交媒体平台的货币化结构导致大约 98% 的创作者收入低于贫困线。

创作者经济中没有中产阶级，98% 的创作者太多了，不容忽视。作为一个 Web 3.0 视频平台，Ziktalk 可以帮助 98% 的用户将其内容货币化。

SocialFi 应用程序以丰富服务不足的人群#

区块链行业的爆炸式增长为那些有能力创造或投资的人创造了巨大的财富，而没有适当基础设施或稳定收入的地区的人口在很大程度上被排除在外。然而，进入印度尼西亚等人口众多的欠发达地区仍有巨大的机会，菲律宾，印度。Ziktalk 的竞争优势在于其低门槛。任何拥有蜂窝设备和互联网连接的人都可以参与社交挖矿并谋生。 任何拥有蜂窝设备的人都可以在 Ziktalk 上获得 ZIK，这是在服务欠缺和人口稠密地区（如印度尼西亚和菲律宾）接触用户方面的巨大竞争优势。

三、ZIK 代币#

什么是 ZIK 代币？#

ZIK 是 Ziktalk 经济的本地货币。无论何时交换价值，ZIK 都会从一个用户转移到另一个用户。

代币详情 代币：Ziktalk 代码（符号）：ZIK 总供应量：10,000,000,000 ZIK 区块链：Luniverse，ERC-20 小数点：18

在交换的情况下，Luniverse 和 ERC-20 网络上发行的 ZIK 将保持 1:1 的比例。ZIK运行在Luuniverse上，所有应用内交易发生在它的侧链上，几乎是即时的，令人毫无感觉。

Ziktalk 区块链#

Ziktalk 网络是由Luuniverse提供支持的区块链。Ziktalk 的主网建立在 Luuniverse 之上，为 Ziktalk 应用程序部署了侧链。Ziktalk 应用程序内的交易发生在侧链上，但是当用户从 Ziktalk 应用程序将 ZIK 提取到外部地址时，令牌在到达接收方地址之前通过 Ziktalk 的主链。 侧链的实施为在 ZIktalk 应用程序中进行交易的任何人提供了免费和即时的交易。只有提款时，用户才需要支付网络服务费。

Zik 代币社交挖矿#

要使用 ZIK，用户必须以一种或另一种方式获得 ZIK 代币。Ziktalk 通过称为社交挖掘的概念引入了一种赚取 ZIK 代币的新方式。社交挖矿是指通过 Ziktalk 上的社交活动赚取 ZIK 的做法。以下是如何挖掘 ZIK 的列表。

注册挖矿 创建一个 Ziktalk 帐户将产生 ZIK 奖励（一次性活动） 推荐挖矿 推荐注册 Ziktalk 的朋友将获得 ZIK 奖励 活动挖掘 在 Ziktalk 上完成应用内活动将获得 ZIK 奖励 社区挖矿 与应用程序上的其他用户互动（喜欢、关注）将产生 ZIK 代币。社区挖矿没有单独的代币池 - 相反，它将是 ZIK 从用户到用户的转移。

将 ZIK 代币供应量的一半分配给社交矿池（“代币分配”部分中的更多代币分配细节），用户将有机会通过简单地使用 Ziktalk 应用程序来赚取 ZIK 代币。即使在社交矿池耗尽的情况下，用户也可以通过各种方式与社区成员互动来挖掘 ZIK。

社交挖矿减半#

减半是指挖矿奖励数量减半的事件。比特币遵循基于区块数的减半时间表，这意味着人们可以在一定程度上预测减半何时发生。 与比特币一样，Ziktalk 将在达到某些里程碑时发生减半事件。然而，与比特币不同的是，当满足用户获取目标时，Ziktalk 社交挖矿就会减半。 在总共 10,000,000,000 个中，有 5,000,000,000 个 ZIK 代币分配给社交矿池。

五、Luniverse区块链#

什么是Luniverse#

Luniverse 是由 Dunamu 的Lambda256开发的基于以太坊的许可区块链。Ziktalk 与 Luniverse 合作，建立了一个独立的主网络，拥有自己的侧链，可在 Ziktalk 应用程序上实现快速、无费用和安全的交易。 Luuniverse 主链管理来自独立链的数据。主代币创建、转账、区块挖矿等区块交易记录保存在主链上。Luuniverse 采用增强型 PoA（权威证明）共识算法进行区块生产。主链选择了 25 个验证者，他们负责推动共识和验证交易。验证者还拥有治理投票权。

Luuniverse的优势#

  • 性能： 1,200 至 2,000 TPS 1秒出块时间 块缓存以读取 TX 可扩展性的链分片
  • 信任： 两层锚固 签名人质押
  • 可用性： 易于访问和使用 没有Gas费 锚定到以太坊网络
  • 安全： DDoS 弹性防御 安全评估
