概述#

有些时候，保持最新状态并回顾加密生态系统可能是一项艰巨的任务，但你仍然可以看到特定项目或行业的叙述和炒作。

在过去的几个月里，ZK项目的资金、开发和应用不断以越来越快的速度浮出水面。许多最聪明、根深蒂固的加密OGs（币圈术语-可理解为圈层–风车注），以及越来越多的杰出CT（加密推特er–风车注）贡献者，都认为，当ZK-rollup（ZKR）技术大规模投入使用时，加密生态系统以及潜在的世界将在技术上取得进步。

虽然这一现实在其全面发展中还需要5-10年的时间，但我们将在未来几个月内开始看到完全兼容的ZKR区块链的应用。

这篇文章旨在回顾最近的发展，绘制当前ZK玩家的立场、他们所构建内容的潜在时间表以及何时发币。很明显我们正在慢慢接近ZK的未来。没有解释区分所强调的协议的技术细节，因为它们需要对数据可用性，有效性和意志验证，安全性和分散性进行细致入微的讨论，以及每个协议采取的其他次要权衡和技术差异。帖子底部有一些链接，其中包括这些深入的讨论以及两个市场领导者zkSync和StarkWare的比较。

ZK概述#

汇总扩展的重点是降低区块链的成本（成本不仅仅包括费用，还有隐私、时间等等–风车注），并且对于将区块链技术（L1不可知）扩展到数亿（可能是数十亿用户）至关重要。虽然使用欺诈证明的Optimistic Rollups现在正在扩展（Arbitrum/Optimism），但使用链下计算的ZK Rollup是我们未来扩展的最佳选择。

你会发现对Rollup技术最深入的了解来自V神，毫不奇怪，他总结道，“从中长期来看，随着ZK-SNARK技术的改进，ZK Rollup将在所有情况下获胜。

ZKR 实现了与Optimistic Rollups相同的目标，同时为用户提供了隐私和提高可扩展性的选项。正如EthHub上所说，“ZK通过减少交易中保存的数据量，减少了用于验证区块的计算和存储资源；对整个数据零知识。归纳来说，ZK技术允许用户证明他们拥有或知道某些东西，而无需提供任何关于所述事物是什么或多少的信息。对于ZK技术来说，随着对加拿大高调的金融平台化以及围绕俄罗斯入侵乌克兰的经济制裁，ZK技术的新热情和对隐私交易的需求而重新燃起。

ZK 近期公告回顾#

据报道，StarkWare以60亿美元估值募集了1亿美元资金，在此之前2021年11月报道募集5千万美元资金估值达到20亿美元。

StarkWare构建的StarkNet于2月22日完成发布，并将在"下半年"转移到社区控制。

Immutable X 以25亿美元的估值筹集2亿美元

GameStop与Immutable X合作建立他们的NFT市场，并宣布为其生态系统提供1亿美元的基金

ZK技术区块链Aleo于2月7日以14.5亿美元的估值筹集了2亿美元

zkSync开发人员Matter Labs于2021年11月筹集了5000万美元，并于2月22日在以太坊的测试网上向公众推出了zkEVM。

Matter Labs和BitDAO宣布提供2亿美元的DAO，以继续为zkSync生态系统进行扩展和核心基础设施建设。

Polygon 以4亿美元的价格收购ZK rollup创业公司Mir Protocol

Polygon此前曾在八月份以2.5亿美元的价格收购了Hermez Network，这是另一个ZK rollup扩展解决方案，合并了代币MATIC和HEZ - 总共6.5亿美元购买了ZK rollup协议。

Polygon最近一轮于2月7日筹集了4.5亿美元，以扩大ETH规模和ZK技术运营

Espresso，ZK技术支持的L1，于3月6日stealth building中脱颖而出并宣布启动3200万美元的资金

Aztec Network在2021年12月筹集了1700万美元的A轮融资

Argent，ETH DeFi移动钱包，推出zkSync驱动的L2钱包，等候名单上有超过50万人

当前参与者

StarkWare — 构建 StarkNet 和 StarkEx

StarkNet

StarkNet是使用ZK-STARKs的ZK-Rollup L2，于11月29日首次登陆ETH主网，该团队一直在不懈地发布新功能和更新。

2月23日，StarkNet宣布了他们的最新版本StarkNet Alpha，并且扩展解决方案已准备好用于应用程序构建和事务。要查看StarkNet生态系统的最新概述，您可以查看他们的网站。

最近，StarkNet宣布与Aave，Opera Browser，DeversiFi和Argent合作，还有无数其他协议等待部署。

何时发币：“在下半年” - 大概是向早期用户空投

StarkEx

StarkEx是通过编程语言Cairo构建的可扩展性引擎，自2020年6月以来一直在主网上运行，目前由DeversiFi，Immutable X，dYdX和Sorare使用。

根据StarkWare的数据，他们从3月9日开始对StarkEx使用量的最后一次更新是11亿美元的TVL，1.23亿笔交易和4.62亿美元的累计价值。

何时发币：“最终由社区运行”，但我的假设是，在StarkNet令牌发布后不久，如上所述，这是"在下半年”。- 再次可能向早期用户空投

zkSync

zkSync是使用ZK-SNARKs的ZK rollup，自2020年6月以来一直上线，由Matter Labs开发。在2月22日的一条推文中，zkSync宣布zkSync 2.0已经上线，并在以太坊的测试网上包含了第一个与EVM兼容的ZK Rollup。

上面链接的推文中的博客文章包括zkEVM里程碑的含义，“去年五月，我们为zkSync的zkEVM制定了我们的愿景，这是一个无需许可的图灵完备ZK rollup，将允许开发人员使用以太坊的原生编程语言Solidity在低费用，高度可扩展的第2层环境中构建和部署分散的应用程序。他们的帖子中有关zkSync 2.0当前功能的更多详细信息。

有关zkSync的更多信息，您可以在此处查看他们使用ZK解决方案构建的项目生态系统，并参考Dune Analytics zkSync看板，其中包含交易，TVL，存款和取款到zkSync以及其他信息的概述

何时发币：根据他们的官方文档，确认令牌将存在，但很难知道何时存在。

Immutable X

$IMX由StarkWare的StarkEx提供支持，是领先的ZK技术支持的协议之一，具有以游戏和NFT为中心的应用程序。$IMX形容自己说：“毫不妥协地为市场，游戏和应用程序体验零汽油费，即时交易和碳中和NFT”。

基于$IMX构建的产品包括Gods Unchained，Illuvium（$ILV）和TikTok的Top Moments系列。

GameStop x Immutable X - 2月3日，GameStop宣布将在$IMX的基础上建立其NFT市场，并宣布设立1亿美元的基金，专门用于支持其生态系统内的游戏开发。GameStop还表示，他们计划将市场纳入"数十亿个可以轻松买卖的低成本游戏内资产”。

$IMX已经构建了多年，并完成了支持 ZK 的 NFT 和未来大规模游戏所需的绝大部分繁重开发和提升。

Polygon （$MATIC）

目前有三个独立的 ZK 汇总扩展计划正在开发

Polygon Miden - “一个基于STARK的zk-Rollup，支持任意智能合约” - 目前正在开发中

Polygon Zero - “提供汇总和验证模式，使用户能够访问更高的吞吐量和更低的费用” - 正在开发中

Polygon Hermez - “一种开源zk-Rollup，针对安全，低成本和可用的令牌转移进行了优化”

Polygon在2021年8月写了一篇很好的博客，讲述了他们对ZK技术的承诺，他们对该行业的10亿美元资金分配，以及对ZK技术的总体概述。

虽然很难获得他们的ZK rollup套件的详细时间表，但很明显，他们致力于通过支持ZK的技术解决方案来扩展ETH，这可以从他们最近价值约6.5亿美元的收购中得到证明。

Espresso Systems

Espresso于3月6日从stealth building中脱颖而出，宣布了资金并介绍了他们的愿景和区块链。

Espresso将自己描述为"一个结合了权益证明共识和ZK rollup机制的第1层区块链系统”，从而产生了"单次扩展和隐私解决方案"。

发布了CAPE（以太坊的可配置资产隐私），这是一个EVM应用程序，允许用户自定义谁可以看到所有权详细信息和资产的移动，并能够在CAPE中包装和解开当前的ERC-20令牌。

CAPE将在"未来几周"部署到Rinkeby测试网，以及有关其ZK扩展解决方案的更多详细信息。

Aztec Network

L2 ZK-Rollup扩展解决方案，“充当ERC20令牌传输和其他DeFi交互的隐私保护”。

zk.money是Aztec构建的核心应用程序，允许MetMask用户与团队秘密发送DAI和ETH，希望"今年晚些时候"获得完整的钱包级资产隐私。

Aztec的L2汇总解决方案使用zk-SNARKs，目前部署在Goerli测试网上

Aleo

Aleo是一个启用ZK的，默认私有的基础层区块链，它"处理链下交易，然后由网络节点验证（而不是执行和重新执行）。

Aleo于2月7日在B轮融资中筹集了2亿美元，他们的区块链目前处于测试网中，计划于今年第三季度在主网上推出。

结论和延伸阅读#

零知识证明处于月球数学，密码学和技术创新的前沿，越来越多地将我们推向全球24/7，隐私支持的廉价交易的未来。目前的两个行业领导者是zkSync和StarkWare（StarkEx / StarkNet），他们都在今年某个时候确认了代币，这是为自己定位的东西。正如其融资轮次和当前估值所证明的那样，需求将是天文数字，但期待已久的L2 szn和下游效应可能最终发生，因为这些协议和其他L2激励用户和流动性将代币的压力转移到彼此身上。虽然我们距离在微观尺度上看到这一点还有几个月的时间，但在宏观上还有几年的时间，但ZK技术可能会改变区块链行业的未来。
