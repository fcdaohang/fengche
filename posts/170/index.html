BAT#

基于区块链的数字广告生态系统的原生货币称为BAT（Basic Attention Token）。
该项目的核心是补贴内容创造者和消费者。
BAT 开发人员解决了在线广告的一系列挑战，包括隐私、广告滥用和安全危险等问题。
现成产品为： Brave 浏览器，你可以在这个浏览器中看到广告，并且登录的用户获得 BAT 代币（随浏览次数）。
使用 BAT 作为支付手段，广告商可以为其营销推广支付费用。
这个代币暂时没有进一步的应用。
即便如此，它还是一种广受欢迎的数字货币，在CoinMarketCap的市值排行榜上排名第 76位。
Binance和Coinbase都将 BAT 列为其交易所的一个选项。
（灰度基金也有其仓位–风车注）

FTM#

Fantom FTM提供智能合约和基础公链服务。
它关注三个方面：可扩展性、安全性和去中心化，同时支持开发新的 DApp 和当前 DApp 的集成。
此外，Fantom 高度重视速度，平均在不到一秒的时间内处理交易。
FTM 是一种用途广泛的多功能 ERC-20 代币。
首先，任何希望运营自己的 PoS 验证节点的人都必须至少拥有一百万个 FTM。
它也被接受为网络支付。
生态系统中的 FTM 持有者也有投票权。
通过质押代币可以获得 14% 的年利率（APR）。
但是，随着质押中玩家数量的增加，收益率会下降。

NOW#

ChangeNOW是一个用于快速加密交换和处理的平台，使用 NOW 作为其原生代币。
超过 300 万人注册了这家快速发展的加密货币初创公司。
ChangeNOW 是一个成熟的加密生态系统，因为它的主要活动是交换货币。
该平台的其他功能包括支持内部加密货币钱包，以及可以轻松执行的空投和代币交换。
NOW 代币的制造者为生态系统的一部分提供了一系列好处。
NOW 平台上的自定义代币列表可以使用平台的 NOW 代币作为支付方式进行支付。
NOW 节点服务，它允许 API 访问完整的节点和区块书浏览器，也为您提供折扣。
NOW 代币还用于支付营销费用，例如 SMM 广告和客座文章。
除了上述应用之外，NOW 还可以用作投资工具。
Binance 和 Uniswap 交易所提供两种 NOW (ERC-20) 之一；另一个，NOW (BEP-2) 在 Binance Chain 上列出。
ChangeNOW 为那些投入资金的人提供每年高达 25% 的显着投资回报率 (ROI)。
此币种暂时只能从DEX交易，走势图参考–风车注
