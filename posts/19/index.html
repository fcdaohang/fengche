关于$UST，一直以来有很多疑惑和困扰占用了我的时间

是时候好好谈谈与其相关的Anchor协议（$ANC）、19.5%年化收益的可持续性、$UST挂钩还有$LUNA价格影响的问题了

本文还将研究Anchor协议的解决方案是如何能以最佳方式处理这些问题的

LUNA为何被各种攻击#

为何有这么多人讨厌$UST和Anchor协议呢？看看这个

一年前$UST市值仅2.6亿美金，Terra（LUNA主链）生态还只有一个协议：Mirror协议。

今天它的市值高达112亿美金，其生态已有17个协议落地（2022年还将增加100个以上项目）

1年增长了43倍

$UST已经超过$DAI甚至目标为稳定币头把交椅

听上去不太可能？也许，但是看到Terra在基于0的基础上如此成功的市场营销，简直太疯狂。

当$UST市值10亿美金时，@stablekwon（Terry联合创始人兼CEO）告诉人们$UST市值在2021年底将达到100亿美金

只有少数人相信他，而他的确兑现了诺言。

那么是什么保持着Terra经济生态的发展呢？

答案是：Anchor Protocol

了解Anchor协议#

Anchor协议大致是一个稳定币储蓄账户，易于使用到一个5岁的孩子也能理解。

以$UST年化19.5%的收益率进行奖励，你就获得了一个很棒的理财产品。

你存入$UST，得到$UST 19.5%的年化收益，就是这样。

几乎所有其他稳定币质押/收益机会要么很难理解很难使用，要么高年化很快就消失。

也就是说Acnhor协议是一个享有19.5%年化收益且收益率永不过期的储蓄产品。

还是不相信为什么这个产品如此成功？

首先，在加密浪潮以及推特上，常见的是了解和使用加密钱包以及各种跨链操作的人，但是请记住，对于现实中大多数人而言，这非常难。

Anchor协议通过提供一种具有19.5%的存款收益率稳定币解决了这个问题并面向股票、债券和房地产的投资竞争。

这吸引了这些传统市场的投资者（相对于加密投资者而言）。

后续会回到稳定保持19.5%年化收益问题

但在此之前，我们必须了解$UST和$LUNA之间的机制和关系。

$UST和$LUNA的机制和关系#

$UST稳定币由另一种资产提供背后支撑-$LUNA。为了保持挂钩，流通中稳定币的供应将被改变以满足需求。

过程？

它是通过$UST和$LUNA之间一个简单的燃烧/铸造机制完成的。

1个$UST可以随时兑换1美元或价值1美元的$LUNA

举个例子：$UST 当前价格 $0.95 (低于挂钩价)

在这个例子中，任何人可用$0.95购买$UST以及燃烧它去换取等值1美元的$LUNA，这将导致$UST重新接近其挂钩价，而且用户获得0.05美元无风险利润。

例子2：$UST 当前价格 $1.05 (高于挂钩价)

在这个例子中，用户可购买价值1美元的$LUNA并燃烧它获得1个$UST（价值$1.05）,结果也将导致$UST重新接近挂钩价，而且用户获得0.05美元无风险利润。

你明白这是如何工作的吗？

当$UST铸造时，对应的$LUNA也会被烧毁。换句话说，随着对$UST的需求上升，$LUNA的供应量就会下降，反之亦然。

2021年秋季，我们看到$UST需求大幅增长。这导致了$LUNA的大幅燃烧，也就是$LUNA的供应减少（通货紧缩）。 由于$LUNA流通较少，加上$UST需求增多，$LUNA的价格便水涨船高。

$UST的市值在3个月内从25亿美金增长至100亿美金，而$LUNA的价格也近乎出现相同的涨幅（大约29美金涨至103美金，并达到ATH年内高点）。

所以，当市场上涨时，一切都很好。系统性风险来自死亡螺旋或银行挤兑。

$UST和$LUNA崩盘的风险#

而当$UST长期跌破挂钩汇率时，以下操作是有利可图的：

  1. 烧掉$UST铸造$LUNA
  2. 卖掉$LUNA获利
  3. 重复以上操作

这将导致$LUNA的价格暴跌，而失去$LUNA价格支撑的$UST将变得一文不值。

让我进一步解释一下：

如果$UST跌破1美元，烧掉他并铸造$LUNA就能卖掉赚钱，这将导致$LUNA的供应量大幅增加，加上卖单压力$LUNA价格自然就会降低（OTC柜台交易不是立即的，而是积累一段时间内的大量交易）。

$LUNA的价格下跌最终导致恐慌，每个人争相恐后出售自己的$LUNA，Terra生态系统中的信任将消失，使$UST变得一文不值。

然而，自21年5月以来，$UST跌破挂钩汇率，人们谈论这种死亡螺旋的风险却从未发生过。

$LUNA是否由可能经历519时期跌至6美元相同的崩盘呢？

自5月以来，Terra生态系统发生了很多事情，让我来解释一下为什么我认为不会再发生这样严重的崩盘。

519时期，$BTC从6万美元下跌到3万美元时，市场出现了很多恐慌。各种山寨币雪崩，$LUNA也不例外。

$LUNA大幅下跌的原因和其他币种类似，它也是因为清算，Anchor协议的清算。

投资者用他们的$LUNA和$ETH借款参与市场，并且没能及时偿还这些贷款。

$LUNA价格随着清算快速下跌->借入资产的抵押品价值下降->清算->$LUNA价格下跌更多->抵押品价值降低->流动性增加->继续下跌…诸如此类。

$UST和$LUNA崩盘会发生吗？#

不会，为什么我认为这不会再次发生？请看：

  1. Kujira Orca: 一种可以从 Anchor协议 购买他人清算资产的协议（类似于不良资产处置） 这将导致代替被清算的$LUNA直接砸盘的风险，有了这个协议，可以经由其用户扮演买家的角色立即购买这些被清算的$LUNA（相当于保护$LUNA的价格-没有流动性投入二级市场）

  2. Whitewhale协议（白鲸协议）：让你我这样的普通散户投资者使用Anchor协议利用套利机会赚钱的协议。 这将保护$UST的汇率挂钩。官方发布了视频解释了协议是如何工作的：https://www.youtube.com/watch?v=ttdYpSmSx4Y（风车注：类似套利机器人，它会自动使用协议资金在$UST汇率脱钩时进行反复的套利交易从而达到获利）

  3. Oracle rewards: 社区池中有14亿美元的$UST和3000万美元的$LUNA代币，将在未来2年内奖励$LUNA的质押者。

质押奖励将使$LUNA的价格越低越具备吸引力，人们会越来越多的购买$LUNA和$UST

价格上涨+更少清算。（上图）

19.5%的高年化可持续吗？#

OK，现在我们应该明白$UST和$LUNA背后的各种关联机制了，可以开始讨论Anchor协议以及可持续的19.5%年化收益率

将Anchor协议看作一家公司。

为了支付19.5%的年化收益率，他们需要收入。

Anchor两条途径赚钱：

  1. 用户贷款的利息收入：11.5%
  2. 抵押品的质押奖励（风车注明：贷款需要抵押品，而链上抵押品是可以立刻质押获利，没有或极低损耗）：bLUNA 8.5%/bETH 4.6%

Anchor的资产负债表：

花费 总存款利息支出：$55亿的$USTx19.5% / 100 = $10亿7千万 收入 借出的利息收入： $UST：$13亿5千万 x 15% / 100 = $2亿 质押奖励 bLUNA：$28亿 x 8.5% / 100 = $2.4亿 质押奖励 bETH：$4.3亿 x 4.6% / 100 = $2千万

余额: -10亿7千万 + 2亿 + 2.4亿 + 2千万 = -$6亿1千万

目前Anchor协议每年有6.1亿美元的赤字（亏）

这相当于每天亏170万美元，这就是为什么自12月$LUNA的年内新高以来收益率储备一直下降很多的原因。

是的，Anchor协议有一个收益储备，可以支付19.5%，直到储备为空。 收益率储备在上个月稳步下降，人们开始担忧。

但请记住，收益率储备是为这样的市场环境而创建的。我们应该感谢甚至存在收益储备。 如果收益率储备归零，这不是世界末日。 APY将根据市场环境进行调整。

现在大约是9-10%，但它会更高（你很快会明白为什么），那么 Terra 如何解决这个问题呢？

  1. @stablekwon 暗示将向收益储备注入 1-3 亿美元（但这只是一个短期解决方案）
  2. 更多 bAsset（风车注：跨链资产） 进入 Terra（bSOL 和 bATOM 已确认，我相信还会有更多）
  3. Anchor V2协议模型：可以添加多个bAssets（例如$SOL或$AVAX），更高的收入和更高的质押奖励
  4. 使用社区池中的 $UST 来增加收益储备（短期）
  5. 限制 Degenbox $UST - $MIM：（这在Wonderland $TIME 的 Sifu 丑闻之后已经得到处理），我认为人们不会继续借用 $MIM，这也消除了对 $UST 的需求。
  6. 限制每个用户的存款（不太现实，因为这会阻止 Terra 生态系统的发展）
  7. 为Anchor协议中的 $ANC-token 做一个用例：比如orion.money或crypto.com，如果你有$ORION或$CRO。和它们相比，你的稳定币投资收益回报更高，为何还去投资不稳定的山寨币呢？所以为什么Anchor协议不能做类似的事情，例如你想要19.5%的高年化，你必须拥有多少数量的$ANC，如果没有，年化收入就降至15%等等。

我确信有人已经写过这方面的文章，但我希望Anchor团队能够更多地探索这个模型。 如果这已经是一个被拒绝的治理提案，我很想知道为什么：D

如你所见，有很多解决方案，我认为Terra团队会解决这个问题。 我唯一担心的是@stablekwon是否选择不重新填充收益储备。 因为，他不必。

但 10-15% 之间的短暂锚定利率可能不会受欢迎。 我相信很多人会从Anchor协议中提出他们的存款并寻求其他收益机会（贪婪本性）。

这样想吧。 11月初，Degenbox策略来了，突然间你可以在$UST稳定币上获得超过100%的年化。人们开始习惯，越来越多的人变得贪婪。

当然，之后人们开始对Degenbox策略失去信任，重新认可Anchor协议的19.5%年化。

但$FTM、$AVAX、$SOL和$ETH等其他公链上仍然有很多收益机会，对投资者非常有吸引力。我认为@stablekwon 无法承担不补充收益储备的机会。

如果锚定率降低，人们将从$UST逃到其他机会（贪婪），这将减少$UST的市值并增加$LUNA供应 = 降低$LUNA价格。

总结和深入探索#

Anchor Protocol 是一个超级成功的借贷协议，也是 Terra 生态系统的基石。我认为 Anchor协议继续照常营业（19.5%年化）对Terra的成功至关重要。
