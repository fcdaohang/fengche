有没有想过加密到底是什么？它与 Web3 有什么关系，它是未来吗？

说实话，这并不难弄清楚。您只需要了解一些 Web3 项目并带着兴趣深入研究就能搞定。
本文中，我将谈论我过去经历过的 5 个 Crypto & Web3 项目以及我对每个项目的看法。
如果您决定坚持下去，这将是一篇有见地的文章。如果还没打算，我希望你以后好奇时再回来看看。

Uhive#

Uhive 并不是一个典型的社交网络，其使命是“为其所有者赚钱”——Uhive 的建立是为了为所有人创造财富。它的目标是每天向所有活跃用户支付其加密货币（数字红利）。

我已经在一定程度上探索了 Uhive，我觉得它是我的日常社交媒体平台，我可以在其中发布和探索更广泛的兴趣。
它在其应用程序中公开了元宇宙。 Oasis 旨在升级为 Web3，这些区块链产品让我兴奋！看到 3D 模型如何真正满足我们的期望是令人信服的。这个产品尤其对我来说是一股清新的空气。

这不仅仅是滚动浏览我的主页，它还穿越不同的空间和社区，从一个帐户滑到另一个帐户（在绿洲中），并与我最感兴趣的内容互动。作为个人，我看到了基于此的潜力设计及其附带的计划。
如上所述，这还是所有人的收入来源之一。现在，与其他社交媒体平台不同，它旨在与所有用户分享其财富资产，作为参与其应用程序的奖励。

到目前为止我在 Uhive 应用程序中创建了 3 个空间，而且一直很享受这些。
我的默认空间是我的第 1 个兴趣（Uhive 有 26 个兴趣）——书籍，我每天都会在其中发布我最喜欢的书籍报信息，并且我每天登录都会获得 HVE2 代币（Uhive 的原生社交代币）。
我还关注与我最喜欢的兴趣相关的许多其他帐户，只是简单地评论其他帐户并喜欢他们的帖子就可以让我在整个平台上得到认可。
这种感觉好像在坐过山车般新奇刺激，这与其他社交媒体所经历的感觉不同。所以，我希望坚持这个，它获得的成功越多，对我来说就越好，因为随着时间的推移，我每天赚取的代币将具有更高的价值。

Ripple#

Ripple 是 XRP 加密货币背后的公司。一种为其数字支付网络构建的技术。
Ripple 的金融技术提供金融交易的实时消息传递、清算和结算。 Ripple 通过 RippleNet 连接银行、支付提供商、数字资产交易所和企业，提供一种无摩擦的全球汇款体验。许多主要银行都使用 XRP 支付系统，例如桑坦德银行（美国）。

XRP 的目的是充当两种货币或网络之间交换的中间机制——作为一种临时结算中介。

我发现在 Ripple 上进行交易的费用也很低，与银行进行跨境支付的高额费用相比，标准 XRP 交易所需的交易成本最低。与比特币不同，比特币可能需要更长的处理时间，而且比特币交易处理费比 XRP 高很多。

我过去使用过 Ripple，因为我在国外时发现它非常有效，当时我我遇到了一些冲突，因此我需要一种更快的交易方法。我想用英镑换欧元。
在这个过程中，我在 Ripple 网络上将 GBP 兑换成 XRP，然后用它们来购买欧元，而不是直接通过银行或货币兑换所处理货币兑换。
这个过程的好处是它更快、更便宜，而不是支付银行和汇款组织可能收取的高额费用。

Stellar#

Stellar 的设计目的是让银行只需支付少量费用就可以实时向世界各地汇款。它提高了金融机构和系统的效率。
另一方面，Stellar 的设计考虑了典型用户和发展中地区的用户。
该平台允许用户以简单、快速、低成本的方式在全球经济中进行交易。
虽然公众可以使用 Ripple，银行可以使用 Stellar，但每个都是为专业消费者打造的，以便为他们的目标市场提供更高效的体验。

Stellar 的目标是通过创建可访问、廉价且易于使用的全球支付系统来增加金融包容性。
想象一下通过一个简单的界面轻松地在朋友之间发送适量的钱来分摊餐费等。这正是我使用基于平台的小额支付服务时提供的。我发现它非常迎合我的想法，因为它的用户可行性和期望是按照我的期望设计的。

另一个需要注意的活动是 Stellar 网络上的每个代币都可以交换为网络上的任何其他代币，并且 Stellar 协议连接买家和卖家。
用户可以通过一个快速的程序将他们的出价或询价发送到公共网络，并且自动解决兼容的交易。没有调解人来促成和解，也没有公司担任中间托管人。
总之，Stellar 的账本系统包括一个去中心化交易所（又名 DEX）。这个平台很有趣，网络保证结算。所以你真的在一个安全的环境中。

Hypto#

听说过这句话，“软件正在吞噬地球。”——Marc Andreessen 的名言。 Well Hypto 认为，金融的吞噬速度似乎慢了十年。为了发展，他们设计了自己的产品，以帮助任何公司成为金融科技公司，这意味着无需与银行进行复杂的集成和文档记录即可管理付款和收款。

我在在线游戏期间使用过 Hypto。同时向我的游戏帐户添加资金以进行商品购买。我在 Hypto 上创建了一个独特的虚拟帐户。
当我开始使用它时，一切都感觉是瞬间的，例如当资金通过 UPI 添加到虚拟账户时收到交易警报。这项活动的好处是，任何商家都可以通过发布 UPI ID 和 QR 码来接收即时交易回调。

这才刚刚开始，并没有得到太多认可，但它绝对是具有潜力的东西。我相信 Hypto 有更大的计划。
生态系统支持的技术演进——Hypto 现在正在分解其原始基础设施和代码方面的财务状况，让每个人都可以使用它。这将使所有开发商和公司能够创建自己最先进的金融产品。

Thirdweb#

这并不是一个大平台，但我在他们为观众设计的项目中发现了一些令人着迷的因素，我忍不住参与其中。他们以其工具及其为您的工作流程带来的开发效率提高而闻名。
Thirdweb 支持区块链、以太坊、Ploygon等多链生态系统。它们提供智能合约、SDK 和 UI 组件，创作者、游戏工作室和开发人员可以将其集成到他们的应用程序中。

我决定想了解更多关于这个平台的信息，在这个平台上，我探索了用户如何创建 NFT 和自定义代币、管理 drop 并在一个应用程序中开发自己的白标市场。
我非常喜欢这个 Web3 软件的一点是工具和 SDK 是能够免费使用的，这吸引了我，因为我不必像使用 Thirdweb 那样担心把钱放在哪里，您只需支付一小部分版税即可。

因此，如果你很懒，想知道一种让你“偷懒”的方法，除此外就别无他法了，因为 Thirdweb 就是专门为此设计的。
它带有放置指南，用户可以在其中发现许多可能性。我发现一个非常有效的方法是带有 Lazy Minting 的 ERC721，它可以指示不同的 NFT 创建，无论是将 React 链接到平台还是通过软件创建基本 NFT 而无需编写代码。它节省了研究时间和从哪里开始的困惑，并且由于他们希望观众注意到团队的支持，我认为拥有带有证据注释的指南对他们成功宣传他们的平台非常有效。

你现在想从某个地方开始吗？还是仍然一无所知？

祝你好运，如果你在整个阅读过程中对任何事情感兴趣，但如果你仍然想了解更多，可以找到这些项目的官方网站去深入探索它们。

总的来说，我觉得我的最爱必须是 Uhive 和 Thirdweb。 Uhive 代币赚取系统对于初学者来说似乎真的很有意义，因为他们不知道从哪里开始，而这个项目初期也没有风险，你可以选择先赚钱，然后当你掌握更多想法时再投资。
Thirdweb 看起来真的很酷，精英会员系统似乎想进入平台并停留一段时间并获得长期投资的回报。 NFT 会员卡肯定有很多优势等等。
